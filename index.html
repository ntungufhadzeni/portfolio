Soccer Bets Project

The "Soccer Bets Project" is a web app developed with Django, Celery, Celery Beat, Redis, and PostgreSQL. It provides soccer enthusiasts with betting tips for upcoming matches. Using Celery and Redis, it delivers real-time updates and employs scheduled tasks through Celery Beat. PostgreSQL stores user data and historical tips. Admins can update tips via the admin dashboard.

Python Django Docker Celery Celery Beat Redis Nginx Postgres
